        <h2>Dicas para Construir Decks</h2>
        <ul>
            <li><strong>Balanceie as cartas:</strong> Inclua unidades terrestres, aéreas e feitiços. Exemplo: Deck "LavaLoon" com Balões, Bebês Dragões e Lava Hound para ataques aéreos.</li>
            <li><strong>Adapte ao meta:</strong> Use decks populares como "Royal Giant" para controle ou "Miner Poison" para defesa. Atualize com base nas mudanças de balanceamento.</li>
            <li><strong>Elixir eficiente:</strong> Priorize cartas com bom custo-benefício, como o Rei Gigante (custo 5) para pressão constante.</li>
        </ul>
    </section>

    <section id="estrategias">
        <h2>Estratégias Gerais</h2>
        <ul>
            <li><strong>Defesa inteligente:</strong> Não gaste elixir desnecessariamente. Use torres para defender e contra-ataque com unidades rápidas como o Cavaleiro.</li>
            <li><strong>Ataque coordenado:</strong> Sincronize cartas, como colocar um tanque (ex.: Golem) seguido de dano (ex.: Feitiço de Fogo).</li>
            <li><strong>Suba de troféu:</strong> Jogue em arenas mais altas para ganhar mais XP. Evite derrotas consecutivas usando decks versáteis.</li>
            <li><strong>Eventos especiais:</strong> Participe de desafios como "Clash Royale League" para recompensas extras.</li>
        </ul>
    </section>

    <section id="dicas-avancadas">
        <h2>Dicas Avançadas</h2>
        <p>Para jogadores experientes: Foque em contra-jogadas. Se o oponente usar um deck de tanques, use feitiços como Veneno ou Tornado para neutralizar. Pratique no modo Treino!</p>
